Software

AnoForPHP: Is a patch for ZendEngine/PHP (5.3.5) to allow native annotations in the language by using the token "@@" (the @ was already taken). It allows 0 or more annotations in classes, methods, properties, and arguments for methods with 0 or more arguments per annotation.

Ding: Ding is a framework that aims to bring to PHP a lot of the available features in the java world, like Dependency Injection (DI), Aspect Orientend Programming (AOP), Inversion of Control (IoC). It is modeled after Spring(tm) from SpringSource. Ding is the winner of the 2011 Binpress programming contest (Read more here.)

PAMI: PHP Asterisk Manager Interface library. Event oriented interface to AMI, that lets you manage your asterisk.

PAGI: PHP Asterisk Gateway Interface library. This high level framework provides the needed abstraction layers on top of AGI, to make telephony applications.

MDM: Marcelog Device Manager is multithreaded daemon and libraries written in plain ansi C, for linux and freebsd, used to communicate with DSLAM Alcatel, ZTE, HUAWEI.

Pakman: A package manager written in plain ansi C for freebsd, uses berkeley db.

If you liked any of the software above (and have a generous soul, of course), please contribute so I can dedicate more of my spare time to it!
