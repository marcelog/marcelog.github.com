Ding: Ding is a framework that aims to bring to PHP a lot of the available features in the java world, like Dependency Injection (DI), Aspect Orientend Programming (AOP), Inversion of Control (IoC). It is modeled after Spring(tm) from SpringSource.
PAMI: PHP Asterisk Manager Interface library. Event oriented interface to AMI, that lets you manage your asterisk.
PAGI: PHP Asterisk Gateway Interface library. This high level framework provides the needed abstraction layers on top of AGI, to make telephony applications.
