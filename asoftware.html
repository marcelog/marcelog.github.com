Ding

  • About
  • Installation
  • Documentation
  • Manual
  • Development

DI ( Dependency Injection ), AOP ( Aspect Oriented Programming ), JSR 250/330 annotations, xml and YAML for bean definitions, lightweight, can be deployed as a PHAR file, simple, and quick MVC ( Model View Controller ), syslog, tcp client and server, with non blocking sockets, timers, and custom error, signal, and exception handling, PAGI Integration (for asterisk gateway interface or agi), PAMI Integration (for asterisk management). Needs PHP 5.3, very similar to seasar, spring ( java ) . Tested with doctrine2! ;) Ding is the winner of the 2011 Binpress programming contest :) Read more here.
